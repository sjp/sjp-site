The GOMstreamer tool is used to allow Windows, OSX and Linux users to watch the GOMtv GSL via VLC. GOMstreamer was originally based on the prior work of Tomas Herman but has been updated to work on newer seasons' streams, along with the addition of new features. Note: This requires a GOMtv account to work and the account must solely authenticate with GOMtv and not via a third party (e.g. Twitter, Facebook).

There are two separate releases of GOMstreamer, one works in OSX and Linux while the other works solely on Windows.

  • Windows
  • OSX
  • Linux

Windows

This release is called GOMstreamer.NET and requires .NET Framework 2.0 (or greater) to be installed. Any computer running XP, Vista or 7 should be able to run this fine.

Download: GOMstreamer.NET can be downloaded from this server. Because this is a binary release, the source has been made available on GitHub.

Usage of GOMstreamer.NET is reasonably straightforward. Enter in the email address and password associated with your GOMtv account and select the location of VLC. If you want to watch the stream, set the mode to 'Play' to get VLC to play the GSL stream. The stream URL will appear in the textbox in case you want to use the stream with some other media player, or to share with others (useful if the GOMtv webserver is down). Unfortunately due to recent changes from GOMtv this mode has been broken (unless someone can suggest a way of piping output from wget.exe to vlc.exe). To play a stream, just use the 'Save' mode and open the file in VLC as it is being saved.

To save the GSL stream for later viewing, follow the same instructions as before but also set the location to save the stream to, then set the mode to 'Save'.

In order to schedule a recording, follow the same instructions as for saving a stream, but set the mode to 'Delayed Save'. This will allow you to set a time at which recording should start. The time is specified in Korean time (HH:MM 24H). GOMstreamer.NET will minimise to the system tray until 1 minute prior to the recording starts in order to stay out of the way.

Note: GOMstreamer.exe must reside in the same directory as wget.exe in order to execute correctly.

OSX

Download: The latest version of GOMstreamer for OSX can be downloaded via GitHub.

To use GOMstreamer, enter your GOMtv email and password into the play.command script, along with the stream quality you wish to use out of SQTest, SQ and HQ (Note: only SQTest will be available unless your GOMtv account has a premium ticket). Execute the script when the GSL stream is live.

To save the stream, enter in the relevant details into the save.command script. This requires setting your username, password, a stream quality, and a location to save to. Execute save.command to begin saving the stream.

Streams can be scheduled to record at a specific time. This is accomplished via the use of the delayed-save.command script. Follow the same instructions as if you were saving the stream, but set the time at which the stream recording should begin in Korean time (HH:MM 24H).

To play the stream that GOMstreamer dumped, open the file (default = dump.ogm) in any decent media player, like VLC. Note, you are able to watch the streams in VLC as they are downloading, though this will require another instance of VLC to be loaded.
