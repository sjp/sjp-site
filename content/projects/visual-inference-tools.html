Along with Garrett Grolemund, Ben Stevenson and under the guidance of Dr. Chris Wild, I helped develop Visual Inference Tools. It is a GUI for R, used for teaching high school and undergraduate students statistical concepts by using animation. These concepts include bootstrapping, randomisation testing and confidence intervals.

I am the current maintainer of this R package. Development occurs on GitHub in the Visual Inference Tools repository.
